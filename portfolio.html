e-Xtract

Mobile Application | Team Project

e-Xtract is a mobile application that has been developed with the help of my friends. It is an app designed to contribute towards decreasing the e-waste output of the Philippines by letting users see the valuable components of their e-waste.

My role in this project was to design the user interface and experience of the application. I also collaborated with my team members to ensure that the designs were implemented correctly in the development phase.

Technologies used: Flutter, YOLOv11, Computer Vision
